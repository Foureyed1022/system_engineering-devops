# 0x02 Shell, I/O Redirections and filters

This directory contains exercises on shell input/output redirections and text filtering commands.

## Tasks

- `0-hello_world` - Print "Hello, World" to standard output.
- `1-confused_smiley` - Print a confused smiley face using redirection.
- `2-hellofile` - Display the contents of a file.
- `3-twofiles` - Display the contents of two files.
- `4-lastlines` - Display the last 10 lines of a file.
- `5-firstlines` - Display the first 10 lines of a file.
- `6-third_line` - Display the third line of a file.
- `7-file` - Create a file with a specific name using redirection.
- `8-cwd_state` - Print the current directory state.
- `9-duplicate_last_line` - Duplicate the last line of a file.
- `10-no_more_js` - Remove JavaScript files from the listing.

## Concepts

These exercises cover:
- Output redirection (>, >>)
- Input redirection (<)
- Command substitution
- Pipes (|)
- Filtering and text processing commands (head, tail, grep, sed, etc.)
