# 0x01 Shell Permissions

This directory contains exercises for working with Linux shell permissions, ownership, and user/group identity.

## Tasks

- `0-iam_betty` - Print the current user using a shell script.
- `1-who_am_i` - Display the effective username.
- `2-groups` - Show the groups a user belongs to.
- `3-new_owner` - Change file owner and group.
- `4-empty` - Create an empty file with specific permissions.
- `5-execute` - Set executable permissions on a script.
- `8-James_Bond` - Apply permissions and ownership for a special file scenario.
- `9-John_Doe` - Apply permissions and ownership for another user scenario.
- `11-directories_permissions` - Set permissions on a directory.
- `12-directory_permissions` - Adjust directory permissions for access control.
- `13-change_group` - Change the group ownership of a file.

## Usage

Each exercise file is designed to be run in a bash shell. Review the comments and expected behavior in each script or command file, then execute it to verify the result.
